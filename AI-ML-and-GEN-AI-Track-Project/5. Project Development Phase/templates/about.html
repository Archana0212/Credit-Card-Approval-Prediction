💳 Credit Card Approval Prediction

This project uses Machine Learning to predict whether a credit card application is likely to be Approved or Rejected based on applicant information.

⚙ How it works

  • User enters applicant details through the prediction form
  • Input data is processed and transformed before prediction
  • Machine Learning model analyzes patterns from previous applications
  • Logistic Regression predicts the approval status
  • System displays prediction result, confidence score, and explanation

🛠 Technologies Used

  • Python - Programming Language
  • Flask - Web Application Framework
  • Machine Learning - Predictive Analysis
  • Logistic Regression - Classification Algorithm
  • HTML & CSS - User Interface Design
