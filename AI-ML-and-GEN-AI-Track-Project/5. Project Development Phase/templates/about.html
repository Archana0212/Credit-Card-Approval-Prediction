About This Project

This application predicts whether a credit card application is likely to be approved or rejected using a machine learning workflow.

How it works

  • User enters the applicant's details in the form.
  • The application validates inputs and transforms them in the same order used during model training.
  • A logistic regression model uses the cleaned input to estimate approval likelihood.
  • The result page shows the decision, a confidence score, and an explanation.

Technologies used

  • Python
  • Flask
  • Scikit-learn (Logistic Regression)
  • HTML, CSS
