# 💳 Credit Card Approval Prediction

A Machine Learning-based web application that predicts whether a credit card application is likely to be **Approved** or **Rejected** based on applicant information. The application uses a trained **Logistic Regression** model integrated with a **Flask** web application to provide real-time predictions along with confidence scores and personalized explanations.

---

# 📌 Project Overview

Credit card approval is an important decision-making process for financial institutions. Manual evaluation of applications is time-consuming and may lead to inconsistencies. This project automates the approval process using Machine Learning by analyzing applicant information and predicting whether the application should be approved or rejected.

The application offers a simple and interactive interface where users can enter applicant details and instantly receive the prediction result along with a confidence score and explanation.

# ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Archana0212/Credit-Card-Approval-Prediction.git
```

### 2. Navigate to the Project Folder

```bash
cd Credit-Card-Approval-Prediction
```

### 3. Install Required Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```
